# 🚀 Proyecto Final

## 🎯 Objetivo

Aplicar todo el flujo profesional del curso en una entrega completa.

## Contexto

Vas a trabajar como si estuvieras resolviendo una tarea real de un equipo. Debes crear una rama, construir un pequeño entregable, registrar los cambios correctamente y abrir un Pull Request bien presentado.

## 🧪 Ejercicio: Perfil profesional

### Tarea

Crear archivo `perfil.md` con:

- nombre
- rol
- descripción

## ⚙️ Flujo obligatorio

1. Crear la rama:

```bash
git switch -c feature/perfil
```

2. Crear el archivo
3. Revisar con `git status`
4. Agregar cambios con `git add .`
5. Hacer commit:

```bash
git commit -m "Agrega perfil profesional"
```

6. Publicar la rama:

```bash
git push origin feature/perfil
```

7. Crear Pull Request
8. Completar la descripción del PR

## 🧑‍🏫 Evaluación

- uso correcto de rama
- commit profesional
- PR claro
- archivo correcto
- orden general del flujo

## Recomendación

No hagas todo de memoria. Usa el curso como guía y demuestra criterio en cada paso.
