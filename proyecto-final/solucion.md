# ✅ Solución sugerida

## Ejemplo de contenido para `perfil.md`

```markdown
# Perfil Profesional

## Nombre
Ana Perez

## Rol
Desarrolladora junior

## Descripcion
Estudiante interesada en Git, GitHub y trabajo colaborativo.
```

## Flujo completo

```bash
git switch -c feature/perfil
git status
git add .
git commit -m "Agrega perfil profesional"
git push origin feature/perfil
```

## Qué hacer en GitHub

1. Entrar al repositorio
2. Abrir Pull Request desde `feature/perfil`
3. Completar el título y la descripción
4. Explicar qué archivo se creó y por qué

## Explicación

- este ejercicio integra ramas, commits, push y PR
- el archivo `perfil.md` funciona como entregable final
- el objetivo no es solo crear el archivo, sino demostrar dominio del flujo completo
