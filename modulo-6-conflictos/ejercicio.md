# 🧪 Ejercicio 6: Resolver un conflicto

## Contexto

Este ejercicio simula que dos versiones distintas del mismo archivo entran en conflicto.

## Tarea

1. Abrir el archivo con conflicto
2. Identificar las dos versiones marcadas por Git
3. Decidir el contenido final
4. Borrar las marcas del conflicto
5. Guardar el archivo
6. Ejecutar:

```bash
git add nombre-del-archivo
git commit -m "Resuelve conflicto en nombre-del-archivo"
```

## Qué se evalúa

- si entendiste qué partes estaban en conflicto
- si el archivo final quedó limpio
- si hiciste el `add` y el commit de resolución
