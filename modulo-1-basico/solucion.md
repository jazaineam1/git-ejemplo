# ✅ Solución sugerida

```bash
touch hola.txt
git status
git add .
git commit -m "Agrega primer archivo de saludo"
```

## Explicación

- `touch hola.txt` crea el archivo
- `git status` te permite verificar que Git detectó el archivo nuevo
- `git add .` lo mueve al staging area
- `git commit -m "Agrega primer archivo de saludo"` crea el primer registro formal del cambio

Si en tu entorno `touch` no está disponible, puedes crear el archivo directamente desde el editor.
