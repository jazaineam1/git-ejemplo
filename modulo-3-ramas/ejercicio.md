# 🧪 Ejercicio 3: Trabajar en una rama

## Contexto

Vas a simular una tarea real: crear una rama para desarrollar un cambio sin tocar `main`.

## Pasos

1. Crear la rama:

```bash
git switch -c feature/perfil
```

2. Crear archivo `perfil.txt`
3. Escribir contenido simple relacionado con un perfil
4. Hacer `git add .`
5. Crear un commit claro
6. Publicar la rama:

```bash
git push origin feature/perfil
```

## Qué se evalúa

- uso correcto de rama
- nombre razonable de la rama
- commit claro
- push exitoso a GitHub
