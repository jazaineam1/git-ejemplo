# ✅ Solución sugerida

## Paso 1. Crear el archivo

Dentro de `mejoras.txt` se puede escribir algo como:

```text
- Mejorar el diseño del login
- Agregar validación de correo
- Optimizar tiempo de carga
```

## Paso 2. Revisar el estado

```bash
git status
```

## Paso 3. Agregar el archivo

```bash
git add .
```

## Paso 4. Hacer el commit

```bash
git commit -m "Agrega lista de mejoras del sistema"
```

## Explicación

- el archivo resume mejoras imaginarias del sistema
- `git status` permite confirmar que Git detectó el cambio
- `git add .` prepara el archivo para el commit
- el mensaje del commit describe con claridad qué se agregó
